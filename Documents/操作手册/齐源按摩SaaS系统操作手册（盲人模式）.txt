齐源按摩 SaaS 系统 操作手册（盲人模式）

本手册面向盲人或低视力的技师、收银员。
全程支持键盘操作和读屏软件朗读，不需要鼠标，也不需要看屏幕。
请用读屏软件（电脑端如 NVDA、读屏精灵；手机端如 VoiceOver、TalkBack）逐行收听。
本手册句子短、步骤少、每步只做一件事。

第一部分 通用说明

一、本手册覆盖哪些端
1. 电脑桌面端（在店里电脑上使用）。
2. 网页端（用浏览器打开店铺网址，操作和桌面端一样）。
3. 微信小程序（在自己手机上使用，技师和顾客都能用）。
电脑端和网页端功能完全一样，下面统称“店端”。

二、开启无障碍模式（电脑端 / 网页端）
1. 输入账号、密码，登录系统。
2. 登录后，焦点移动到界面右上角，找到“无障碍”开关。
3. 在开关上按回车或空格，打开无障碍模式。
4. 打开后：字号会放大，焦点边框会加粗，读屏朗读更清晰。
5. 这个设置会被记住，下次登录还是开着的。

三、键盘怎么用（电脑端 / 网页端通用）
1. 按 Tab 键，焦点向后移动到下一个控件。
2. 按 Shift 加 Tab，焦点向前移动到上一个控件。
3. 每移动到一个控件，读屏都会朗读它的名称。
4. 在按钮上，按回车或空格，执行这个按钮。
5. 在下拉框上，按上下方向键选择，按回车确认。
6. 在输入框里，直接打字输入。
7. 在列表里，用上下方向键逐项移动，读屏会逐项朗读。
8. 任何时候按 F1，都会打开使用帮助。

四、用快捷键直接进功能（电脑端，不用找菜单）
按下后，系统会语音播报已经进入的功能名称。
看不到某个菜单，说明当前账号没有这个权限，对应快捷键也不生效。
   F2 进入收银台。
   F3 进入预约管理。
   F4 进入订单流水。
   F5 进入房间管理。
   F6 进入会员管理。
   F7 进入会员类型。
   F8 进入技师排队。
   F9 进入日报与业绩。
   F10 进入日结交班。
   F11 进入服务项目。
   F12 进入优惠券。
   Ctrl 加 F 进入数据看板。
   Ctrl 加 Q 进入物耗库存。
   Ctrl 加 W 进入服务评价。
   Ctrl 加 E 进入投诉处理。
   Ctrl 加 R 进入排班与请假。
   Ctrl 加 T 进入提成规则。
   Ctrl 加 Y 进入工资结算。
   Ctrl 加 U 进入员工管理。
   Ctrl 加 I 进入门店管理。
   Ctrl 加 P 进入服务订阅。

五、金额怎么朗读
1. 系统朗读金额时，按“元、角、分”朗读，不用小数点。
2. 例如 128 元 5 角，不会读成“128 点 5”，避免听错。
3. 听金额时，请留意“元、角、分”三个字。

六、不同角色能做什么
1. 技师：主要用“技师排队”，上钟、下钟，查看本人业绩和工资。
2. 收银员：用收银台开单、结账，管理预约、会员、房间。
3. 店长、店主：在收银员功能之外，还能管理员工、排班、提成、工资、门店、订阅。
说明：你能看到哪些功能，由你的角色决定，看不到的就是没有权限。

第二部分 技师常用操作（手机小程序为主）
一、登录小程序
1. 在手机微信里打开“齐源按摩”小程序。
2. 打开手机读屏（VoiceOver 或 TalkBack）。
3. 焦点移动到登录入口，按提示输入账号、密码登录。
4. 登录后进入技师首页。

二、技师首页能做什么
1. 首页有：今日上钟状态、今日业绩、本月业绩、待办提示。
2. 用手指逐项滑动，读屏会逐项朗读。
3. 首页下方有底部标签，可切换到不同功能页。

三、上钟和下钟（技师排队）
1. 在店端电脑上：按 F8 进入“技师排队”。
2. 用 Tab 找到“上钟”或“下钟”按钮。
3. 在按钮上按回车，切换自己的状态。
4. 上钟表示开始接客，下钟表示暂停接客。
5. 切换后读屏会朗读当前状态。

四、查看本人业绩
1. 小程序里进入“业绩”页，或店端按 F9 进入“日报与业绩”。
2. 读屏会朗读今日业绩、本月业绩。
3. 金额按“元、角、分”朗读。
4. 可以听到接钟数量、点钟数量、提成金额。

五、查看本人评价
1. 小程序里进入“评价”页。
2. 读屏会逐条朗读顾客给的评分和评语。
3. 可以了解顾客的满意度和建议。

六、查看本人工资
1. 小程序里进入“工资”页。
2. 读屏会朗读本月应发工资、提成、扣款等。
3. 如对工资有疑问，请联系店长或店主。

七、计时房（如门店使用）
1. 小程序里进入“计时房”页。
2. 读屏会朗读当前计时房的开始时间和已用时长。
3. 按提示操作开始或结束计时。

第三部分 收银员常用操作（店端无障碍模式）

一、进入收银台
1. 在店端电脑上按 F2，进入收银台。
2. 读屏会播报“已进入收银台”。

二、开单（加入服务）
1. 按 Tab 移动到服务项目列表。
2. 用上下方向键选择要做的服务，读屏会朗读服务名称和价格。
3. 在选中的服务上按回车，把它加入购物车。
4. 重复以上步骤，加入多个服务。

三、设置技师和点钟
1. 在购物车里的服务项上，找到“技师”下拉框。
2. 用方向键选择技师，回车确认。
3. 找到“点钟 / 轮钟”开关，按提示选择。
4. 如需修改数量或时长，在对应输入框里输入。

四、关联会员
1. 按 Tab 移动到“会员手机号”输入框。
2. 输入会员手机号，回车检索。
3. 读屏会朗读检索到的会员姓名和卡内余额。
4. 如门店配了磁条读卡器，也可以直接刷会员卡调出会员。

五、结账
1. 按 Ctrl 加 S，进入结账。
2. 按提示选择支付方式：现金、会员卡、微信、支付宝、银行卡。
3. 用方向键选择支付方式，回车确认。
4. 如需记小费，在小费输入框里输入金额。
5. 确认后完成结账，可选择打印小票。
6. 结账成功后读屏会朗读本单金额。

六、新开一单
1. 在收银台里按 Ctrl 加 N，新开一单。
2. 之前的购物车会清空，开始新的开单。

七、预约、会员、房间
1. 按 F3 进入预约管理：新增、查看、修改顾客预约。
2. 按 F6 进入会员管理：查询会员、开卡、充值、查看消费记录。
3. 按 F5 进入房间管理：查看房间、开计时房、结束计时。
每个页面都用 Tab 移动焦点，按回车操作，读屏会逐项朗读。

第四部分 常见问题

1. 听到“订阅已到期，仅支持只读”。
   说明门店订阅过期了，现在只能查询、不能保存。
   请联系店主，在“服务订阅”里续费。续费后恢复正常。

2. 进入某个功能没有反应，或没有内容。
   可能当前账号没有这个功能的权限。
   请联系店长或店主确认权限。

3. 读屏没有朗读，或朗读不清楚。
   先确认无障碍模式已经打开（界面右上角的无障碍开关）。
   再确认读屏软件已经启动。

4. 金额听不清楚。
   金额按“元、角、分”朗读，请留意这三个字。
   可以让读屏重复朗读当前焦点。

5. 找不到某个按钮。
   用 Tab 或 Shift 加 Tab 逐个移动焦点，读屏会逐项朗读名称。
   也可以用本手册第一部分的快捷键直接进功能。

6. 需要人工帮助。
   请联系平台客服。客服电话和微信在“服务订阅”页可以查到。
