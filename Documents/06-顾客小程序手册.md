# 顾客小程序手册

> 适用对象：来店消费的顾客。
> 系统入口：微信搜索店铺名（如「明心盲人按摩」）→ 进入小程序。
> 顾客**不需要事先注册账号** —— 全程靠微信身份 + 手机号。
>
> 配套文档：《03-店主店长手册.md》。

## 1. 你能用小程序做什么

| 你想做的事 | 在小程序里点哪 |
|---|---|
| 预约到店时间 | 首页 → 「预约到店（顾客）」 |
| 查我什么时候到 | 首页 → 「我的预约（顾客）」 |
| 绑定会员卡，接收充值/到期/生日提醒 | 我的预约页 → 「绑定会员卡」 |
| 看会员卡余额 / 在用套餐 | 我的预约页（绑定后自动显示） |
| 取消预约 | 我的预约 → 选一条 → 「取消预约」 |

> 收银结账还得来门店，小程序不收钱。

## 2. 第一次用：3 步走

### 2.1 进小程序

- 方法 A：扫店里收银台贴的小程序码
- 方法 B：微信「发现」→ 小程序 → 搜店铺名
- 方法 C：朋友/技师转发小程序卡片给你

### 2.2 选门店

如果该店有多家分店，首页会提示选门店。一旦选定后续都默认这家——想换在「首页」可以再选。

### 2.3 授权（可选但推荐）

- 「订阅消息授权」：弹窗时选「总是允许」—— 会员卡到期 / 生日 / 充值都能在微信收到
- 「手机号授权」：在「我的预约」绑会员卡时使用

## 3. 预约到店

**首页 → 「预约到店（顾客）」**：

1. 选门店
2. 填姓名
3. 填手机号（11 位）
4. 选服务项目（如「中式按摩 60 分钟」）
5. 选指定技师（也可选「不指定」让店里轮号）
6. 选到店日期（手指点日历）
7. 选到店时间（手指点时分）
8. 选人数（1-5 人）
9. 备注：偏好 / 健康注意（如「腰椎不好，力道轻」「孕妇」「精油过敏」）
10. 提交预约

> 备注写清楚很重要——技师做按摩前会看「上钟前必读」，你的健康提示就在那里。

提交成功会跳「我的预约」列表，可以看到刚提交的这一条。

## 4. 我的预约

**首页 → 「我的预约（顾客）」**：

1. 输入手机号 → 「查询我的预约」→ 列出该手机号下所有预约
2. 每条预约显示：门店、到店时间、人数、状态、是否可取消
3. 「取消预约」按钮：到店前 2 小时以上可取消，到点了就不能取消（爽约会累计在会员档案）

## 5. 绑定会员卡

**「我的预约」页面 → 「绑定会员卡（接收微信通知）」**：

- 输入手机号后点绑定 → 系统按你的手机号匹配会员卡（前提是去年充值时登记过这个手机号）
- 绑定成功后：
  - 卡内余额、等级、卡号即时显示
  - 之后充值到账 / 生日 / 卡到期前 30 天 都会推送微信消息

> 一个手机号对应一张会员卡。换手机号了要去门店让收银员改卡上的手机号，否则你绑不上。

## 6. 微信通知会收到什么

绑定会员卡 + 授权订阅消息后，你会收到 4 类提醒：

| 类型 | 什么时候来 |
|---|---|
| 会员卡到期提醒 | 卡到期前 30 / 15 / 7 / 1 天各一次 |
| 生日提醒 | 生日当天 |
| 预约到店前 30 分钟提醒 | 预约的当天，到店前半小时 |
| 充值到账 | 你给会员卡充值后 1-2 分钟 |

通知里会显示：你的姓名、金额（如有）、到期日期。文字本身不含敏感数据。

## 7. 会员套餐展示

绑定会员卡后，「我的预约」会显示你"在用的套餐"：

- 套餐名（如「10 次中式按摩」）
- 类型（计次卡 / 期限卡 / 折扣卡）
- 适用服务
- 剩余（如「剩 3 次」「剩 12 天」）
- 有效期

下次到店收银员结账时会自动选用套餐里的次数（不扣余额）。

## 8. 客服与投诉

小程序里**没有**在线客服按钮（店里小、人不多）。有事直接打门店电话或到店找前台：

- 商品 / 服务质量问题：去门店当面跟前台说
- 隐私问题（不希望接收提醒）：「我的预约」→「解除本机显示」可解绑当前手机的会员卡微信通知

## 9. 常见疑问

### 9.1 我没注册账号怎么有"我的预约"？

预约是按"手机号 + 微信"识别的——你提交预约时填的手机号就是身份。

### 9.2 我不绑卡能预约吗？

能。预约不要求绑卡。绑卡只是为了接收微信通知。

### 9.3 同一手机号多次预约怎么办？

都会出现在「我的预约」里。可以分别处理。

### 9.4 我换手机号了怎么办？

去门店让收银员改卡上的手机号。然后用新手机号重新绑。

### 9.5 我能不能在小程序里付款？

暂时不行。结账要到门店（现金 / 微信扫码 / 支付宝 / 会员卡 / 团购券都行）。小程序只管预约和提醒。

### 9.6 我能看到技师 / 价格吗？

「预约到店」页里有服务下拉 / 技师下拉，可以提前选好——但价格在门店收银时确认（避免改价后旧预约价格错乱）。

## 10. 无障碍

本小程序整体按盲人无障碍标准做的（主要服务盲人技师），但顾客端同样适用：

- 所有按钮、输入框都带 `aria-label`
- 字号 ≥34rpx，配合微信"特大字号"足够清晰
- 金额会用「壹仟元整」朗读
- 单列布局，读屏不跳

不会用智能手机的长辈完全可以让家人代为预约，预约人姓名/手机填实际到店人即可。

## 11. 安全

- 你的手机号、姓名、健康备注**只有这家店的店员能看到**——平台方、其他门店看不见
- 微信"订阅消息"不会泄露你的卡号 / 余额给第三方
- 退出小程序时数据不会留在手机里
