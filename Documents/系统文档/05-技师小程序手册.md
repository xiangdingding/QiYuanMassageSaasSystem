# 技师小程序手册（含盲人无障碍）

> 适用对象：按摩店技师（Technician 角色），尤其面向盲人/低视力技师。
> 系统入口：微信小程序「XX 盲人按摩」（搜店铺名）。
> 默认账号：由店主在「员工管理」创建并下发用户名 / 临时密码。
>
> 配套文档：《03-店主店长手册.md》《07-常见问题与故障处理.md》。

## 1. 小程序对你做了什么

技师每天用 4 个页面：

| 页面 | 路径 | 做什么 |
|---|---|---|
| 我的班次 | `pages/tech/home` | 上钟 / 休息 / 下班；看当前服务房间；上钟前必读 |
| 我的业绩 | `pages/tech/performance` | 今日 / 本月 营业额 + 提成 |
| 我的评价 | `pages/tech/reviews` | 顾客给我的评价 + 平均分 |
| 我的工资 | `pages/tech/payroll` | 历史工资单（封盘后可见） |

进入路径：首页（`pages/index`） → 「我的班次（技师）」/ 「我的业绩（技师）」按钮。

## 2. 盲人无障碍：先做这 3 件事

### 2.1 开启读屏

- **iOS**：设置 → 辅助功能 → 开 VoiceOver。或三击侧键启用。
- **Android**：设置 → 无障碍 → 开 TalkBack。
- 读屏开了之后，所有按钮、文本、输入框都会被朗读。

### 2.2 调大字号（视力较弱时）

- 微信里：我 → 设置 → 通用 → 字体大小 → 选「特大」或「超大」
- 小程序内已使用 ≥34rpx 字号，配合系统字号放大后基本可读

### 2.3 关掉省电模式 / 让屏幕常亮

否则上钟时屏幕黑了，按"语音播报"也不响。

## 3. 我的班次（最重要的页面）

### 3.1 一进页面读屏会朗读

- "今日状态：上钟" / "休息" / "下班"
- "今日已上钟：N 次"
- "当前服务房间：A1 号房 / 项目：中式按摩 60 分钟"（如有）

### 3.2 上钟前必读（黄色警告区）

- 当前客人有健康问题 / 偏好时，页面顶部会显示「⚠ 上钟前必读」卡片
- 内容包括：客人姓名、健康注意（如「高血压、勿按头部」）、偏好（如「不要油，力道中等」）
- 「**语音朗读**」按钮：一键朗读全部
- 「**我已知晓**」按钮：确认后这条必读卡片不再阻挡你；不点不影响工作但会记录"未确认"

> ⚠ 上钟前必读是**强制流程**——客人投诉「我跟他说了过敏没听」时这条记录可以保护技师。

### 3.3 上钟 / 休息 / 下班

三个大按钮，按钮长按读屏会朗读说明：

| 按钮 | 作用 |
|---|---|
| 开始上钟 | 进入接客排队队列，可被叫号 |
| 休息 | 暂停被叫号；如厕 / 吃饭 / 喝水时点这个 |
| 下班 | 离开队列；下班后当日不再被叫号 |

按下后会有语音确认（如"已切换到上钟"），同时收银员的「技师排队」实时刷新。

### 3.4 "语音播报"按钮

一键朗读你当前的状态 + 今日已上钟次数 + 当前服务房间。盲人技师走到收银台前点一下确认状态，比盯屏幕快。

## 4. 我的业绩

### 4.1 看到什么

- 今日业绩：当日营业额（你做的部分）
- 今日提成：当日已下发提成
- 本月累计：本月营业额

金额会用「壹仟贰佰元整」这种朗读友好的方式输出（避免读屏读"1200.00"成"一千二百点零零"），见 `yuanToReadable()`。

### 4.2 "语音播报业绩"按钮

一键朗读"今日营业额 X 元，提成 Y 元，本月累计 Z 元"。

## 5. 我的评价

- 顶部显示平均分（如「平均 4.8 分 / 共 32 条评价」）
- 列表每条评价：顾客昵称、星级、标签（如「手法好」「轻柔」）、评论、时间
- 列表内容会被读屏朗读

差评（≤2 星）会自动进店长的「投诉处理」队列——你不用主动找店长解释，店长会找你了解情况。

## 6. 我的工资

### 6.1 显示规则

只显示「已封盘」的月度工资单。当月还没结算的工资在「业绩」里看预估提成。

### 6.2 一条工资单的构成

```
到手 ¥XXX 元
基础 ¥X 提成 ¥Y 加班 ¥Z 满勤 ¥W 调整 ¥V
小费 ¥T（不入工资）
本月 N 钟  排班 M 天  请假 P 天
备注：…
```

- "小费"单列，不并入"到手"（行业规定，小费现金/即时支付给技师不走对公）
- 每条工资单底部有「语音朗读」按钮

### 6.3 "我对工资单有疑问"

工资封盘后**店长不能在小程序里改**——找店长当面或电话沟通。店长在 BS 端「工资结算」里可以加调整项（下月生效）。

## 7. 通知（订阅消息授权）

绑定后微信可以收到：

- 充值到账（你给客人充值时，客人收到。你自己不会因此被打扰）
- 会员到期提醒（你的会员客户即将到期）
- 预约到店前 30 分钟（你今晚有客）

**首次进入小程序时**会弹「订阅消息授权」——选「总是允许」省事。

## 8. 触屏手势（VoiceOver / TalkBack）

| 手势 | 作用 |
|---|---|
| 单指右滑 | 跳到下一个元素 |
| 单指左滑 | 跳到上一个元素 |
| 单指双击 | 激活当前元素（点按） |
| 双指上滑 | 从顶部连续朗读全屏 |
| 双指轻触 | 暂停 / 继续朗读 |
| 三指上滑 | 滚动 |

> 长按按钮**不**触发动作——盲人技师常误以为长按是确认；本小程序的按钮全部是"双击"触发。

## 9. 单列线性布局

所有页面都是**自上而下一列**，没有左右拼版、没有横向滚动。读屏顺序就是你看到的顺序，不会跳。

## 10. 常见操作速查

| 我想… | 怎么做 |
|---|---|
| 上钟 | 我的班次 → 开始上钟 |
| 休息一下 | 我的班次 → 休息 |
| 下班 | 我的班次 → 下班 |
| 查今天做了多少 | 我的业绩 → 语音播报业绩 |
| 看顾客评价 | 我的评价 |
| 查上月工资 | 我的工资 → 选月份 |
| 申请请假 | 暂不支持小程序自助；找店长 |
| 换密码 | 暂不支持小程序自助；找店长 |

## 11. 异常处理

### 11.1 看不到自己

到了店里发现小程序里"我的班次"显示"账号未绑定"——找店长，可能是店长建了员工档案但没给你绑账号。

### 11.2 读屏不读

- 检查 VoiceOver/TalkBack 是不是开着
- 重启微信
- 仍不行：找店长，店长找平台运维（可能是页面 a11y 属性有 bug，会上报开发）

### 11.3 上钟点了没反应

- 检查网络（小程序需联网）
- 看页面上方有没有"加载中"字样
- 仍不行：人工去收银台跟收银员说"我上钟了"，收银员能在「技师排队」里替你切状态

### 11.4 工资跟我算的差很多

- 自己先按"底薪 + 提成 + 奖金 + 加班 + 满勤 − 调整"对一次
- 找店长，店长查「提成规则」是不是当月生效
- 还有疑问，店长会查 `audit_logs` 给你看每一项的来源

## 12. 隐私

- 你在小程序看的所有数据**只是你自己的**——同事的工资、评价、业绩你看不到
- 别人看不到你的微信账号 / 手机号（除非店长在 BS 端里查）
- 退出登录：「首页」→ 「登录 / 切换账号」→ 点退出

## 13. 给店长的提示（盲人技师就业保障）

> 这一节给店长看。

- 让盲人技师**单独**用一部手机/账号，不要跟家人共用，避免读屏环境被打乱
- 排班时考虑技师"钟数上限"，过载提醒触发时**优先让其他人接**（保护技师手腕/腰部）
- 工资单备注里写中文短句而不是英文/字母——读屏读英文容易拗口
- 用「订阅消息」推送代替短信——盲人技师手机里短信通知声音容易跟其他混
