# 收银员手册（CS 端 Windows 桌面）

> 适用对象：按摩店收银员（Cashier）。店主/店长也能用，菜单完全一样。
> 系统入口：桌面图标「按摩店收银系统」（CS 端）—— 或 BS 端 Web `https://shop.your-domain.com`。**CS 与 BS 功能完全一致**，但 CS 端能直连打印机、钱箱、客显、来电盒、磁条卡，桌面流程更顺手。
> 默认账号：由店主在「员工管理」创建并下发。
>
> 配套文档：《03-店主店长手册.md》《07-常见问题与故障处理.md》。

## 1. 你能看到的菜单（15 项）

收银员看到的菜单是店主菜单的子集——去掉了「服务项目 / 排班 / 提成 / 工资 / 员工 / 门店 / 订阅」。剩下：

收银台、预约管理、订单流水、房间管理、会员管理、技师排队、日报与业绩、日结/交班、会员套餐、优惠券、物耗库存、服务评价、投诉处理。

## 2. 登录与初次配置

1. 双击桌面图标 → 输入用户名 / 密码。
2. 首次登录立即改密（右上头像 → 个人设置）。
3. 收银电脑应已由运维配好打印机、钱箱、客显、来电盒、磁条卡——若刚到的新机器，按《01-上线实施流程手册.md》第 6 章配 `appsettings.json`。

## 3. 全键盘快捷键

CS 端做了全键盘可达 + 自定义快捷键，盲人收银员/老员工尤其依赖（屏幕阅读器无障碍）。

| 快捷键 | 动作 |
|---|---|
| `Alt + 1..9` | 跳转到侧栏第 1..9 项菜单 |
| `F2` | 跳到会员管理（开新卡 / 查卡常用） |
| `Ctrl + N` | 新订单（跳到收银台并开始开单） |
| `Ctrl + S` 或 `Ctrl + Enter` | 立即结账（在收银台时）|
| `F5` | 刷新当前页面 |

> 自定义键位在 `MainWindow.xaml` 里，店主想改让开发支持，**不要自己改 xaml 文件**。

## 4. 收银台（最常用）

收银台一屏完成：选服务 → 指派技师 → 处理会员 → 选支付方式 → 结账打小票。

### 4.1 一笔典型订单

1. **选服务**：搜索框输服务编码（如 `Z60`）→ 列表立即过滤 → 双击加入购物车，或按 `Enter` 加首个匹配。
2. **指派技师**：购物车每一行下拉选技师；不选会跳「请为所有项目指派技师」。
   - 「轮号」按钮：自动取「技师排队」队首。
   - 「指定」按钮：弹列表选具体技师。
   - 同一订单多个服务可分别指派给不同技师。
3. **分配房间**：可选；如门店设了「计时房」则会强制选房。
4. **会员**（可选）：
   - 手输会员手机/卡号 → 「查询」按钮（或 `F2` 跳会员页查后回来）
   - **磁条卡刷卡**：在收银台界面焦点状态刷卡 → 自动调出会员（前提 `appsettings.json` 配了 `cardReader.enabled = true`）
   - 关联会员后单价自动切到会员价；会员等级折扣自动叠加
5. **支付方式**：现金 / 微信 / 支付宝 / 会员卡 / 银行卡 / 团购券。
   - 现金：输实收金额 → 系统算找零（同时显示在客显）
   - 微信/支付宝：扫客人付款码 OR 让客人扫你出的二维码（视前台流程）
   - 会员卡：余额不足会拒绝；如要部分会员卡 + 部分现金，分两笔
   - 团购券：扫券码或手输 → 校验后自动抵扣
6. **小费**（可选）：在结账前输入 → 不入营业额、单独记到技师账户
7. **结账**：`Ctrl + S` 或点「结账」→ 后端创建订单 → 打印小票 + 弹钱箱（仅现金）+ 客显显示实收 + 房间转占用 + 自动通知排队技师

### 4.2 快速开单（适合熟手 / 高峰期）

1. 焦点在服务搜索框
2. 输入「Z60」→ `Enter` 加入
3. `Tab` 到技师下拉 → 输入技师名前两字 → `Enter`
4. `Ctrl + S` 结账

### 4.3 多人单 / 多服务单

直接在购物车里继续加项目即可——一张订单里可以多服务、多技师、多房间。结账时统一收一次钱。

### 4.4 测试打印（现场调试用）

收银台右上「测试打印小票」按钮：发送一张样张 + 踢一次钱箱。打印机刚装好或纸卷换完用它确认 OK。失败会弹窗显示原因（端口被占用 / 找不到串口 / 网络不通）。

## 5. 加钟 / 转钟 / 并钟（接客流核心）

### 5.1 加钟

客人正在做的服务时间到了想再加：

1. **订单流水** → 找到这一单 → 「加钟」
2. 选追加的服务（如再 30 分钟） → 选是否同一技师
3. 确认 → 原订单追加一行 → 房间继续占用
4. 结账时整单一起收钱

> 加钟期间技师不能被排队叫走——「技师排队」会自动锁住 OnDuty 状态。

### 5.2 转钟

技师中途有事换人：

1. **订单流水** → 找到这一单 → 选行 → 「转钟」
2. 选目标技师
3. 提成按"已服务时长"切分给两位技师（系统自动算）

### 5.3 并钟

A 单和 B 单都是同一组客人，决定合并：

1. **订单流水** → 多选两单 → 「并钟」
2. 选保留哪一单的会员关联
3. 合并为一张订单，提成各算各的

### 5.4 投诉改派

客人对当前技师不满意，要求换：

1. **订单流水** → 选行 → 「投诉改派」
2. 填投诉原因 → 选新技师
3. 系统进「投诉处理」队列
4. 提成切分：原技师只拿已服务部分；新技师拿剩余部分。如果完全免除原技师责任，店长可手工 0 计

## 6. 反结账（谨慎！）

误结账要回滚：

1. **订单流水** → 选行 → 「反结账」
2. 输入原因 + 店长密码（如配了）
3. 系统回滚：
   - 订单状态回 `Open`
   - 库存补回
   - 会员卡余额补回
   - 团购券核销解除
   - 提成扣除
   - 已封盘的工资单**不变**（避免重算）

> 反结账操作有审计，店长能看到操作人/时间/原因。

## 7. 会员管理

收银员能做的会员操作：开卡 / 充值 / 查询 / 转赠 / 退卡。

### 7.1 开卡

**会员管理 → 新建** → 填基本信息 + 初始充值（可选）。

### 7.2 充值

会员详情 → 「充值」→ 选金额 + 支付方式。系统自动叠加套餐里配的「充 X 送 Y」。

### 7.3 团购券 / 会员套餐

- 团购券核销：结账时勾「团购券」→ 输入券码 → 自动校验 + 抵扣
- 会员套餐使用：会员有「计次卡」时，结账时自动选用（扣 1 次而不是扣余额）；会员有多个套餐时弹选择框

## 8. 房间管理

- **状态**：空闲 / 占用 / 清洁中
- 开单选房后→ 占用
- 结账后→ 自动「清洁中」（保洁手动 → 空闲）
- 计时房：开始→计时→结束→按小时单价计费→进当日营业额「计时房收入」单列

## 9. 技师排队

- 技师状态：上钟（OnDuty）/ 休息 / 下班 / 请假 / 培训
- 顺序按"上次下钟时间"，最久没上的优先
- 收银台「轮号」就是从这里取队首
- 「指定」可以越过排队让客人指定的技师上
- 技师当日已上钟数接近上限时（默认 12 钟）排队页会黄色告警

## 10. 日报 / 日结 / 交班

### 10.1 日报（实时）

**日报与业绩 → 日报 Tab**：

- 营业额（不含小费、不含反结账单）
- 客流（订单数 / 人头数）
- 通道明细（现金 / 微信 / 支付宝 / 会员卡 / 银行卡 / 团购券）

### 10.2 日结（封盘）

每天打烊前：

1. **日结/交班** → 「开始日结」
2. 系统列出当日所有现金收入 / 找零
3. 你数实际钱箱里的现金 → 输入「实点现金」
4. 系统比对差额：
   - 长款（+）：稀有，挂账
   - 短款（−）：默认从你工资里扣，店长可改为「门店承担」
5. 打印「日结单」→ 店长签字盖章

### 10.3 交班

中途换班（如白班→晚班）：

1. **日结/交班** → 「交班」
2. 跟日结类似，比对你这班的现金
3. 接班人确认 + 双方签字 → 生成「交班单」打印

## 11. 物耗库存

- 进货：物耗库存 → 「进货」→ 选物料 + 数量 + 单价
- 自动扣库存：服务项目里勾了「关联物料」时，每结一单自动扣
- 库存预警：低于安全库存 → 列表红色 + 「物耗库存」菜单右上角红点
- 盘点：「盘点」按钮 → 输入实际数量 → 系统记差异（损耗）

## 12. 服务评价 / 投诉处理

收银员能看不能改：

- 顾客小程序提交的评价 / 投诉自动进对应列表
- 你可以「转交店长」让店长处理
- 自己也可以代客录入投诉

## 13. 预约管理

接电话预约的常规流程：

1. **预约管理 → 新建**
2. 填到店时间、姓名、手机、人数、服务、指定技师（如有）、备注
3. 客人到店当天：列表里点「到店」→ 自动跳到收银台预填好 → 走开单流程

如果顾客在小程序自助预约，记录也会进这个列表。

## 14. 五件硬件每天检查

每天开班前 30 秒确认：

| 设备 | 怎么验 |
|---|---|
| 打印机 | 收银台 → 「测试打印小票」 → 出纸 + 切纸 + 钱箱弹出三件齐全 |
| 钱箱 | 同上 |
| 客显 | 开机后客显有"欢迎光临"；点任意服务到购物车，客显应跳"应付 ¥xxx" |
| 来电盒 | 让同事用手机打一下店里的座机 → 应能听到"来电，号码 138..." |
| 磁条读卡器 | 测试卡刷一下 → 收银台焦点状态下应自动调出会员 |

任何一项失败查《07-常见问题与故障处理.md》或叫店长。

## 15. 多店收银

若你登的是店长账号且管多家店，顶栏「门店」下拉切换。切换后所有数据按目标店刷新。**收银员账号通常绑定单店，无此下拉。**

## 16. 网络断了怎么办

CS 端不带离线缓存——网络断了暂时无法开单。临时处理：

1. 手写收据给客人
2. 网络恢复后用「订单流水 → 补录历史订单」录入

> 不要让网络问题阻塞客人结账——先收钱（手记），后补单。

## 17. 上下班动作

### 上班
1. 启动 CS 端 → 登录
2. 检查 5 件硬件（见 14）
3. 查看「预约管理」当天预约 → 准备好接客

### 下班
1. 日结 → 交班
2. 把现金清点入保险柜
3. 退出登录（右上头像 → 退出）
4. **不要直接关电脑而不退出**——避免下一班用你的账号操作留下错责
